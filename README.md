# EM-Tech
## 이엠텍 (상장사)

### 📌 회사 개요

- **사업 분야**: 전자담배, 미용기기, 의료기기 제조  
- **상장 여부**: 코스닥 상장사  
- **해외 사업장**: 베트남, 인도네시아 등 글로벌 생산 거점 운영

- **임직원 수**: 약 450명

- .NET Framework 환경에서 애플리케이션 설계 및 구현

#### 2. 데이터베이스 연동
- MSSQL 기반 데이터 처리 및 관리
- 생산 데이터 저장, 조회 및 분석 기능 구현

#### 3. 시스템 운영 및 개선
- 생산 현장 이슈 대응 및 기능 개선
- 공정 효율화를 위한 시스템 최적화
